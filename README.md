# Ofrinio Beach Holiday Site

Static promotional website for the Ofrinio Beach apartment.

Open locally:

```powershell
Start-Process .\holiday-site\index.html
```

The site embeds the existing calendar from:

```text
https://martin9020.github.io/calendar/
```

No files in the calendar repository are changed.

For free hosting, upload the `holiday-site` folder contents to GitHub Pages, Netlify, Cloudflare Pages, or Vercel as a static site.
